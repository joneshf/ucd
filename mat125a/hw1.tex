\newcommand\epsdelta[5]{
  We want to prove:
  \[
    \lim_{x \to #1} #2 = #3
  \]

  \begin{proof}
    Given $\epsilon > 0$,
    we want to find $\delta > 0$ such that

    \[
      0 < \left|x - #1\right| < \delta \implies \left|#2 - #3\right| < \epsilon
    \].

    #5

    So, choose $\delta = #4$.

    Then we have
    \[
      0 < \left|x - #1\right| < \delta \implies \left|#2 - #3\right| < \epsilon
    \]
    as was to be shown.
  \end{proof}
}

  \epsdelta{#1}{#2}{#3}{#4}{
    We can simplify the consequent a bit.

    #5

    If we notice, this is exactly the form of the antecedent,
    assuming $\delta = #4$.
  }
}
